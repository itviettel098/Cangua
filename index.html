Văn bản – highlight từng câu
Nhập văn bản phía trên rồi nhấn Đọc để bắt đầu.
Cách hoạt động
1
Nhập API key ElevenLabs trong tab Cài đặt.
Đăng ký miễn phí tại elevenlabs.io → Profile → API Keys
2
Ghi âm giọng nói của bạn (tối thiểu 1 phút, nói rõ ràng).
Chất lượng ghi âm càng cao → nhân bản càng giống.
3
Nhân bản giọng nói – tải lên ElevenLabs và tạo voice ID.
Chỉ mất ~30 giây.
4
Quay lại tab Đọc văn bản, chọn ElevenLabs → giọng của bạn sẽ xuất hiện đầu danh sách.
Nhấn Đọc để nghe giọng mình đọc văn bản!
Bước 1 – Ghi âm giọng nói
Đọc to, rõ ràng khoảng 1–3 phút. Có thể đọc bất kỳ văn bản nào. Nhiều mẫu → kết quả tốt hơn.
Chưa ghi
Chưa có bản ghi âm nào.
Bước 2 – Nhân bản giọng nói
Cần nhập ElevenLabs API key trong tab Cài đặt trước.
Cần ghi âm ít nhất 1 bản ở bước trên trước.
Giọng nhân bản đã tạo
ElevenLabs API Key
Đăng ký tài khoản miễn phí tại elevenlabs.io → vào Profile Settings → API Keys → sao chép key dán vào đây.
Cài thêm giọng tiếng Việt (trình duyệt)

🪟 Windows: Cài đặt → Thời gian & Ngôn ngữ → Giọng nói → Thêm giọng → chọn Tiếng Việt

🍎 macOS: System Settings → Accessibility → Spoken Content → System Voice

📱 iOS: Cài đặt → Trợ năng → Nội dung đọc → Giọng nói

🤖 Android: Cài đặt → Trợ năng → Chuyển văn bản thành giọng nói
