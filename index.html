Chưa kết nối

Điều khiển máy từ xa

Nhập địa chỉ máy chủ VNC + port WebSocket rồi nhấn Kết nối.

1
Cài websockify trên máy cần điều khiển:
websockify --web . 6080 localhost:5900
2
Đảm bảo VNC Server đang chạy (port 5900) và mở firewall port 6080.
3
Nhập host + port vào thanh trên, điền mật khẩu nếu cần, rồi Kết nối.
